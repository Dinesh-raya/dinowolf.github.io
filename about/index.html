About the Creator

Data analyst · Engineer · Builder

Hi, I'm Dinesh Raya. I build interactive tools and resources that make data analytics skills accessible to everyone.

This platform grew from a simple idea: the best way to learn SQL, Excel, and Python is by doing — with real problems, real data, and instant feedback. Every project on this site runs entirely in your browser, so you can start learning without installing anything.

What You'll Find Here
  • SQL Project — 50 YouTube analytics challenges from basic SELECTs to CTEs and window functions, validated in-browser with SQLite
  • Excel Project — 50 formula challenges across 28 categories, from SUM to SUMPRODUCT, evaluated live with HyperFormula
  • Python Project — 50 coding challenges covering basics, pandas, and numpy, running on Pyodide (CPython in the browser)
  • Cheatsheets — Quick-reference guides covering 250+ formulas across SQL, Excel, Power BI, and Tableau
Tech Stack
  • Jekyll + GitHub Pages for static hosting
  • SQL.js (WebAssembly SQLite), Pyodide (WebAssembly CPython), HyperFormula
  • CodeMirror for the editor, lunr.js for full-text search
  • Decap CMS for content management
